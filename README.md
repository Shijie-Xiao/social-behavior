# Social Behavior

Social attention model for multi-agent trajectory prediction with interpretable dual-stream attention.

Predicts future keypoint trajectories for interacting mice using a graph-based recurrent architecture with score-clamped Bahdanau attention. Achieves 12.33 px cbADE on MABe 2022 Mouse Triplets with zero attention degradation.

See [srnn-pytorch/README.md](srnn-pytorch/README.md) for architecture, results, and training instructions.
